- Si vous avez démarré le serveur avec `npm run dev` (ou `.\start-dev.ps1`), la page ci-dessous s'affichera. - Si le serveur n'est pas lancé, ouvrez `index.html` ou lancez `.\start-dev.ps1` depuis le dossier du projet.

Ouvrez le fichier index.html dans le dossier du projet pour un aperçu statique minimal.

Set-Location -Path 'd:\Projets code\Site version 2\ala_deux'
